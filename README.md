# Prime or not
## About

Quick IOS mini game created for class purpose.

## Features

- **Random Number Generation**: Numbers between 2 and 100 are displayed randomly
- **User Interaction**: Tap button-style options to answer (PRIME or NOT PRIME)
- **Instant Feedback**: 
  - Green checkmark (✓) for correct answers
  - Red cross (✗) for wrong answers
- **Track your Results**: 
  - Records correct and wrong answers in real-time with card-style display
  - After **10 rounds**, check your results in the popup and keep going.
- **Timer System**: 
  - 5-second countdown for each number
  - Automatic timeout counts as a wrong answer
  - New number appears automatically after each attempt
